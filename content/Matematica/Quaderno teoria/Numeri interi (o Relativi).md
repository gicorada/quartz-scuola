$\mathbb{Z} = \{\dotso, -3, -2, -1, 0, +1, +2, +3, +100, \dotso\}$
$\mathbb{Z}$ = Insieme dei numeri Relativi o Interi

$\mathbb{Z}$ contiene $\mathbb{N}$
$|\mathbb{Z}| = |\mathbb{N}|$ perché $+\infty = +\infty$

$-2, -3 \rightarrow$ concordi
$+2, -3 \rightarrow$ discordi
$-2, +2 \rightarrow$ opposti

$-2 < -1$
$+2 < +3$

## Somma algebrica
### Concordi
$(+2) + (+3) = +5$
$(-2) + (-3) = -5$
Somma dei valori assoluti, segno concorde
### Discordi
$(+2)+(-3) = -1$
$(-2)+(+3) = +1$
Sottrazione dei valori assoluti, segno concorde al valore assoluto maggiore

## Prodotto/divisione (regole dei segni)
### Concordi
$(+2) * (+3) = +6$
$(-2) * (-3) = +6$
### Discordi
$(+2) * (-3) = -6$
$(-2) * (+3) = -6$

## Potenza
### Base positiva
$2^1 = 2; 2^2 = 4; 2^3 = 8$
Sempre positivo
### Base negativa
$-2^1 = -2; -2^2 = 4; -2^3 = -8$
Positivo se esponente pari

## Esponente pari e dispari
## Esponente pari
L'output è sempre positivo
### Esponente dispari
Il segno dell'output è uguale a quello di partenza

Definizione: $(2)^3 = 2 * 2 * 2 = 8$ (2=base, 3=esponente)

Casi particolari:
- $1^3 = 1 * 1 * 1 = 1$ cioè $1^n = 1$ sempre
- $0^3 = 0 * 0 * 0 = 0$ cioè $0^n = 0$ se $n \neq 0$ 

## Operazioni
### Prodotto con la stessa base
Sommo gli esponenti:
$2^3 * 2^4 = (2 * 2 * 2) * (2 * 2 * 2) = 2^{3 + 4}  = 2^7$

### Divisione con la stessa base
Sottraggo gli esponenti
$\dfrac{2^5}{2^2} = \dfrac{2 * 2 * 2 * 2 * 2}{2 * 2} = 2^{5 - 2} = 2^3$

### Potenza di potenza
Moltiplico gli esponenti
$(2^3)^2 = (2^3) * (2^3) = 2^{3 + 3} = 2^{3 * 2} = 2^6$

### Stesso esponente
$(2^5) * (3^5) = (2 * 3)^5 = (6)^5$ (moltiplico basi)
$\dfrac{2^5}{3^5} = (\dfrac{2}{3})^5$ (divido basi)

### Esponente negativo
$(2)^{-3} = \dfrac{2^3}{2^6} = \dfrac{2 * 2 * 2}{2 * 2 * 2 * 2 * 2 * 2} = \dfrac{1}{2^3} = \dfrac{1^3}{2^3} = (\dfrac{1}{2})^3$

### Esponente zero
$(2)^0 = \dfrac{2^3}{2^3} = \dfrac{1}{1} = 1$ quindi $n^0 = 1$

### Zero alla zero
$0^0 = \dfrac{0^5}{0^5} = \dfrac{0}{0} = indeterminato$
